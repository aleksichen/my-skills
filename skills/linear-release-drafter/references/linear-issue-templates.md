# Linear Issue Templates (Pine)

## Bug Capture Template

```markdown
## Symptom
<what happened>

## Reproduction
1. <step>
2. <step>

## Expected vs Actual
- Expected: <expected>
- Actual: <actual>
```

## Spike Capture Template

```markdown
## Goal
<what to evaluate>

## Scope
- In: <in-scope>
- Out: <out-of-scope>

## Exit Criteria
- <criterion 1>
- <criterion 2>
```

## Completion Comment Template

```markdown
## Delivery Evidence
- Merge/Landing: <link or commit>
- Verification: <commands/results>
- Docs: <updated path>

## For Bug Only
- Root cause: <summary>
- Fix strategy: <summary>
- Regression test: <what was added/updated>
```

## Release Draft Section Template

```markdown
## <version-or-cycle>

### Added
- <item>

### Fixed
- <item>

### Debt/Refactor
- <item>
```
